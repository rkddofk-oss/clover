## 사용 방법

1. 위의 Raw HTML 링크를 브라우저 주소창에 붙여넣으면 HTML 파일이 렌더링되어 웹페이지처럼 열립니다.
2. GitHub Pages를 사용하면 아래의 Pages URL로 바로 접속할 수 있습니다.
3. 로컬에서 확인하려면 파일을 다운로드한 뒤 로컬 브라우저로 열어보세요.
4. 정식 도메인/호스팅으로 공개하려면 Netlify, Vercel 등의 정적 호스팅을 사용하세요.

### GitHub Pages 활성화(간단 가이드)

1. 저장소의 Settings → Pages로 이동합니다.
2. "Source"에서 브랜치를 `main`과 `/ (root)`로 선택한 뒤 "Save"를 클릭합니다.
3. 잠시 후(수 분 내) 페이지 상단에 배포된 사이트의 URL이 표시됩니다. 보통 `https://{사용자명}.github.io/{저장소명}/...` 형태입니다.

> 참고: Pages 활성화 후 자동으로 https://rkddofk-oss.github.io/clover/christmas-in-myeongdong.html 로 접근 가능합니다. HTML 파일이 main 브랜치 루트에 위치해야 하며, 다른 경로에 있으면 URL 경로를 조정해야 합니다.
